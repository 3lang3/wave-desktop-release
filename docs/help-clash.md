# 登陆相关问题

## 代理软件问题

### Mac

- [ClashX]用户需要打开`增强模式`

### Windows

- [Clash for Windows] 需要打开`TUN Mode`

> 开启 `TUN Mode` 需要先安装 `Service Mode`, 如果开启后依然无法使用, 请尝试去设置中开启 `Lightweight Mode`

`Service Mode` 旁边地球标志为绿色即为开启成功
